The Diner is a text-based, perma-death game build in React with an integrated chat lobby. Players connect, give their name, pick from randomly generated stats, then join the chat lobby. From there, players can communicate with each other normally with with "/me" emotes. Players can challenge each other to a fight, with the loser being ejected from the app.
